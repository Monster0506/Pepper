Programming Language Documentation

Learn about the syntax and features of our custom programming language.

Introduction

This project is a custom programming language that uses Reverse Polish Notation (RPN) for arithmetic operations, as well as unique syntax for defining and calling functions, managing variables, and handling conditional statements.

Function Definitions and Calls

Defining a Function

Functions are defined using the :: syntax, followed by a list of parameters and a body of code. 

printHello::()->
    SHOW("Hello from function!")
<-
void

Output: Hello from function!

Calling a Function

Functions are called using the <_ |> FuncName syntax. This will invoke the function with no parameters. 

_ |> printHello

Output: Hello from function!

Function with Parameters

Functions can take parameters and be called with arguments.

greet::(name:string)->
    SHOW("Hello, " + name + "!")
<-
void

("Alice") |> greet

Output: Hello, Alice!

Note: Parameters are passed within parentheses and can be of any valid type such as string, int, etc.

Reverse Polish Notation (RPN) Arithmetic

Basic Arithmetic (RPN)

Arithmetic operations in this language use Reverse Polish Notation. The operands come first, followed by the operator. 

LET x: int = 10
LET y: int = 5
REAS x = x y +  %% 10 + 5
SHOW(x)

Output: 15

Chaining Operations

Operations can be chained together. For example:

LET z: int = 5
REAS z = z 3 + 2 *  %% (5 + 3) * 2
SHOW(z)

Output: 16

Note: You can chain operations using RPN, as the result of one operation is immediately available for use in the next.

Conditional Statements

Simple IF Statement

LET x: int = 10
IF x > 5 DO
    SHOW("x is greater than 5")
END

Output: x is greater than 5

IF-ELSE Statement

IF x < 15 DO
    SHOW("x is less than 15")
ELSE
    SHOW("x is greater than or equal to 15")
END

Output: x is less than 15

IF-ELIF-ELSE Statement

LET score: int = 85
IF score > 90 DO
    SHOW("Grade: A")
ELIF score > 80 DO
    SHOW("Grade: B")
ELIF score > 70 DO
    SHOW("Grade: C")
ELSE
    SHOW("Grade: D")
END

Output: Grade: B

Variable Declaration and Basic Operations

Integer Variables

LET x: int = 10
LET y: int = 5
SHOW(x)
SHOW(y)

Output: 10, 5

Basic Arithmetic with REAS

REAS x = x y +  %% x = 10 + 5
SHOW(x)

Output: 15
