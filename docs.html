Function Definitions and Calls

Functions are a core part of the language. Functions are defined using the :: syntax and are called using the <_ |> FuncName syntax. 

printHello::()->
    SHOW("Hello from function!")
<-
void

Output: Hello from function!

_ |> printHello

Output: Hello from function!

greet::(name:string)->
    SHOW("Hello, " + name + "!")
<-
void

("Alice") |> greet

Output: Hello, Alice!

Note: You can pass arguments to functions within parentheses, separated by commas. Functions can return values as well.

Reverse Polish Notation (RPN) Arithmetic

Arithmetic operations in this language use Reverse Polish Notation. The operands precede the operator. 

LET x: int = 10
LET y: int = 5
REAS x = x y +  %% 10 + 5
SHOW(x)

Output: 15

LET z: int = 5
REAS z = z 3 + 2 *  %% (5 + 3) * 2
SHOW(z)

Output: 16

Note: RPN operations are evaluated from left to right. The result of one operation becomes the input for the next.

Conditional Statements

Use IF-ELSE and IF-ELIF-ELSE to control the flow of your program based on conditions. 

LET x: int = 10
IF x > 5 DO
    SHOW("x is greater than 5")
END

Output: x is greater than 5

Variable Declarations and Basic Operations

Variables can be declared using the LET keyword, and operations are performed using the REAS keyword. 

LET x: int = 10
LET y: int = 5
SHOW(x)
SHOW(y)

Output: 10, 5
